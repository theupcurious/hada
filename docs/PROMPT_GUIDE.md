# Hada Prompt Guide 🚀

Maximize your productivity with Hada by using agentic prompts that trigger multi-step workflows, proactive scheduling, and collaborative document creation.

## 🌟 The "Project Launch" Power Prompt
Use this to kick off any new initiative. It combines research, co-authoring, and scheduling in one turn.

> **"I want to start a new project called '[Project Name]'. Research the top 3 trends in [Field] for this year, then create a 'Project Roadmap' document for me in the '[Project Name]' folder and provide a link to it. Finally, set a reminder for me to review this tomorrow morning."**

---

## 🎨 Hada Workspace & Documents
Hada doesn't just chat; it creates durable artifacts in your workspace.

### Drafting Reports
> "Research the competitive landscape for [Industry] and create a 'Market Analysis' document with a link. Use a professional tone and include a table for competitor comparisons."

### Iterative Editing
> "Read my 'Q2 Priorities' document and update the 'Marketing' section to include a new goal for social media growth."

---

## 🛡️ Proactive Time Defense
Hada helps you protect your most valuable resource: focus time.

### The Focus Protector
> "Look at my calendar for next week. I need to finish the [Project Name] proposal. Use **Time Defense** to identify low-priority meetings we can move and block out 4 hours of deep work time."

### Conflict Resolution
> "Check my schedule and task list for today. Give me a briefing on any potential conflicts and suggest a plan to maximize my output before 3 PM."

---

## 🔌 Advanced Integrations (MCP)
Extend Hada's capabilities with the Model Context Protocol.

### Local File Management
> "Use my local MCP server at http://localhost:3001 to list the files in my 'Documents' folder. Find the most recent markdown file and summarize it."

### Ecosystem Bridge
> "Connect to my MCP GitHub server and list the open issues in my main repository. Create a 'Bug Triage' document with a suggested priority list."

---

## 🧠 Agent Self-Learning
Help Hada become more personalized by maintaining a "Self-Learning Loop."

### Workflow Optimization
> "Based on our last few chats about my writing style, update your **'Self-Learning Log'** with my preference for active voice and short, punchy sentences."

---

## 💡 Tips for Better Prompts
1. **Be Multi-Step**: Hada is built to plan and execute complex sequences. Don't be afraid to ask for 3-4 related tasks in one prompt.
2. **Name Your Folders**: Hada will automatically organize documents into project-specific folders if you specify the project name in your prompt.
3. **Reference Workspace Context**: You can say "Read [Document Title]" to give Hada context from any document in your `/docs` library.
4. **Use Personas**: Switch Hada's style in Settings (e.g., to "Academic" or "Concise") or specify it in your prompt for specific tasks.
